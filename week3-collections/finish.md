## Récapitulatif : Collections

### List
```dart
List<String> items = ['A', 'B'];
items.add('C');
items[0];  // Accès index
items.length;
```

### Map
```dart
Map<String, int> data = {'A': 1};
data['B'] = 2;
data['A'];  // Accès clé
```

### Set
```dart
Set<String> unique = {'A', 'B', 'A'};  // {A, B}
unique.add('C');
```

### Méthodes
```dart
list.where((x) => x > 5);
list.map((x) => x * 2);
list.any((x) => x > 10);
list.every((x) => x > 0);
```
